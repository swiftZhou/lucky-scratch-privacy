## GitHub Pages setup

**Settings → Pages**

- Source: **Deploy from a branch**
- Branch: **`main`**
- Folder: **`/ (root)`**
- Save

Site files live at the **repository root** (`index.html`, `support.html`, `privacy/`, `css/`).

After each `git push` to `main`, wait 1–3 minutes for the site to update.
